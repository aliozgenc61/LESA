# LESA Minimum İşçilik v4

Bu sürüm özellikle klasör yapısı bozulmasın diye **yalnızca kök dizinde 5 dosya** içerir.

Dosyalar:
- `index.html`
- `styles.css`
- `app.js`
- `SUPABASE_SCHEMA.sql`
- `KURULUM_REHBERI.txt`

Kurulum özeti:
1. `SUPABASE_SCHEMA.sql` içeriğini Supabase SQL Editor'de çalıştırın.
2. Supabase Authentication > Users bölümünden ilk kullanıcıyı oluşturun.
3. Bu 5 dosyayı GitHub LESA deposuna yükleyin.
4. Vercel'de GitHub deposunu Import edin.
5. Yayındaki sitede Supabase URL ve anon key'i bir defa girin.
